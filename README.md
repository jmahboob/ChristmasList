# ChristmasList
A small Flask + AngularJS based Christmas list application for my family to use this holiday season.

# Dependencies (so far)

## Python

- Flask >= 0.11.1
- SQLAlchemy >= 1.0.15

## JavaScript/CSS

- Bootstrap >= 3.2.0
- jQuery >= 2.2.1
- AngularJS >= 1.2.21 < 2.0.0
