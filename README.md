# ChristmasList
A small Flask + AngularJS based Christmas list application for my family to use this holiday season.

# Dependencies (so far)

## Python

- Flask >= 0.11.1
- Flask-Login >= 0.3.2
- Flask-WTF >= 0.12
- Flask-bcrypt >= 0.7.1
- Flask-SQLAlchemy >= 2.1
- SQLAlchemy >= 1.0.15

## JavaScript/CSS

- Bootstrap CSS >= 3.2.0
- Angular Bootstrap >= ???
- jQuery >= 2.2.1
- AngularJS >= 1.5.8 < 2.0.0
- Angular-Animate >= 1.5.8 < 2.0.0
- Angular-Smart-Table >= ???
